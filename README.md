# 法院空间下载小助手

一键批量下载中国法院网络服务平台（zxfw.court.gov.cn）案件附件的 Chrome 扩展。
> 号外号外 **现在有粉嫩版本！**
> 建议下载这个分支体验一下哈哈！！
> 👉 [点我跳转齐妃体验版](https://github.com/jmyi-jas/case-space-helper/tree/%E7%B2%89%E8%89%B2%E5%A8%87%E5%AB%A9%E4%BD%A0%E5%87%A0%E5%B2%81%E4%BA%86) 🎀



## 功能（超好用der！！）
- 🔑 自动捕获认证参数，无需手动复制 Token
- 📋 自动获取全部案件附件列表
- ☑️ 支持勾选部分文件下载
- 📁 直接保存到本地文件夹（可选择）
- 🖱️ 可拖拽悬浮面板

## 安装方法（宝宝巴士启动）

1. 点code然后下载压缩件

2. 打开 Chrome，地址栏输入：
   ```
   chrome://extensions/
   ```

3. 右上角开启「开发者模式」

4. 点击「加载已解压的扩展程序」→ 选择项目文件夹

5. 完成！

## 使用方法

1. 打开 [法院网络服务平台](https://zxfw.court.gov.cn) 并登录
2. 进入案件详情页
3. 页面右下角出现悬浮面板
 - <img width="453" height="355" alt="image" src="https://github.com/user-attachments/assets/b1084541-9a51-49d7-8259-e276163bfd9d" />
 - ⬆️大概就是这样
4. 等待四个参数自动变绿（滚动消息列表触发加载）
5. 点击「获取文件列表」
6. 勾选需要下载的文件
7. 点击「下载已选文件」→ 选择保存文件夹

## 注意事项

- 仅支持 Chrome / Edge 浏览器
- 需要已登录法院网络服务平台
- Token 有时效性，过期后刷新页面重新捕获
- 本工具仅供当事人下载自己案件的合法材料使用
